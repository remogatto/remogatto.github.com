L'insegnante di sostegno

E' un insegnante specializzato, previsto dalla Legge 517/77, che viene assegnato, in piena contitolarità con gli altri docenti, alla classe in cui è inserito il soggetto portatore di handicap per attuare «forme di integrazione a favore degli alunni portatori di handicap» e «realizzare interventi individualizzati in relazione alle esigenze dei singoli alunni»

In altre parole, l'insegnante di sostegno è, a tutti gli effetti, un insegnante

  • di classe
  • nella classe
  • per la classe

Ed è - naturalmente - l'insegnante di riferimento per l'alunno disabile.

Di classe

L'insegnante di sostegno è un insegnante di classe e quindi:

  • Firma sul registro di classe
  • Può comminare sansioni disciplinari a tutti gli studenti della classe
  • Partecipa ai Consigli di classe e ha diritto di voto sulle valutazioni di tutti gli studenti
  • Può essere una figura specializzata di supporto agli alunni con Bisogni Educativi Speciali (BES)

Nella classe

L'insegnante di sostegno è nella classe e perciò

  • Quando è in classe lavora in compresenza con l'insegnante curricolare
  • La sua presenza non è neutra e modifica il campo
  • Può essere fisicamente dislocato in qualunque punto della classe
  • E' in un punto di osservazione privilegiato (in posizione terza) e percepisce dinamiche che - il più delle volte - sono nascoste agli occhi degli studenti e dell'insegnante curricolare

Per la classe

L'insegnante di sostegno rappresenta una risorsa professionale aggiuntiva e quindi

  • Mette a disposizione le sue competenze a tutti gli alunni della classe
  • Può essere di supporto al docente curricolare nelle attività didattiche in cui quest'ultimo decida di coinvolgerlo

Per l'alunno disabile

L'insegnante di sostegno lavora a stretto contatto con l'alunno disabile ed in particolare:

  • Lavora con lui/lei in rapporto 1:1, dentro e fuori dalla classe
  • Lavora con lui/lei all'interno di piccoli gruppi di studenti
  • Semplifica il contenuto didattico delle lezioni e a volte lo differenzia
  • Sviluppa e progetta la programmazione didattica dell'alunno con la collaborazione dei docenti curricolari
  • E' in relazione diretta con i genitori dell'alunno/alunna

...ma attenzione!

Questo non significa che

  • l'alunno disabile è di esclusiva competenza dell'insegnante di sostegno
  • la programmazione didattica dell'alunno disabile è materia esclusiva dell'insegnante di sostegno

Insomma, l'insegnante di sostegno è per l'alunno disabile ma tutti i docenti della classe sono ugualmente responsabili del percorso educativo e didattico di quest'ultimo.

Navigate With History

Flowtime.js offers a full support for the HTML5 History API where available or gracefully degrades on the hashchange event.

This means that you can navigate using the browser’s back and forward buttons and deeplink a page for sharing purposes.

Progress Indicator

You can enable a default progress indicator useful not only to know what’s the current page you are looking, but also as a navigation tool.

Look at the bottom right corner and you can see a miniature of the presentation structure.

Clicking on a page thumb will navigate to that page.

Fragments Support

The built in fragments navigation allows to advance step by step inside a page.
Press down to try.

You can discover single elements or even a single part of an element, one at a time.

Fragments navigation is deeply customizable with some configuration options
You can learn how in the documentation.

Fragments Unleashed

Fragments can be more than just delayed piece of page.

Thanks to some special classes you can add a couple of useful custom behaviours.

  • The .step fragment will partially fades out;
  • The .shy fragment will completely hides itself;
  • You can use this two special fragment types to easily create special effects.

Theme and Styling

If you don’t like the default theme or you want to build your own (or both things),
or if you want to build a website on top of Flowtime.js,
you can write your own theme and replace the default one.

Theme and core css are in separate files so you can’t break the layout (unless you override some classes).

Fluid Layout

Everything can be fluid, just use em, rem and % units if you want to make an element resizable.

Native Parallax Support

By popular demand Flowtime.js includes native parallax support*.

Simply add a parallax class to anything you want to have parallax enabled and configure the amount of distance for all elements or for a single element using data-parallax attribute.

For an example go to the next page; for more info read the documentation.

* By the way: I’m not a parallax fan; but ehi… this is the "web 3.0".
